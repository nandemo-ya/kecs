## Features

### Service Management
- Scale services up/down
- Update service configuration
- Force new deployment
- Stop service (set desired count to 0)

### Monitoring
- Real-time task status
- Deployment progress
- Health check status
- Event history

### Task Operations
- View task details
- Stop individual tasks
- View container logs
- SSH into containers (if enabled)

## Keyboard Shortcuts

- `s` - Scale service
- `u` - Update service
- `d` - Delete service
- `f` - Force new deployment
- `l` - View logs
- `t` - View tasks
- `e` - View events
- `/` - Filter services
